Hardware Requirements Selection Tool

Before I built this tool, we were using a 36-page Word file to provide hardware specifications. When that information needed to be handed to a customer, engineers would have to manually edit the Word file as necessary.

This selection tool serves two purposes: 1) to provide information in a user-friendly way, and 2) as a tool to generate professional, customized PDFs that can be delivered to customers.

NOTE: In this demo version, all internal information, including product names, has been altered.

TOOLS: MindTouch, jsPDF, jsPDF-AutoTable, and multiselect.js
